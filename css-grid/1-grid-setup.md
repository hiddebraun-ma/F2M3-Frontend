### Hulp

Hier vind je stukjes code die je kunt gebruiken in je oplossing.
Je moet zelf bedenken waar ze moeten komen. **Gebruik de video hierbij!**.

#### Grid container

```html
<div id="dashboard"></div>
```

#### Grid items

